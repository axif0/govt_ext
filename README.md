# Teletalk Form Fill

Free Chrome extension — save Teletalk job form once, **Auto Fill** on any `*.teletalk.com.bd` site. No subscription. Data stays on your device only.

## Install

1. Chrome → `chrome://extensions` → **Developer mode** ON
2. **Load unpacked** → select this folder

## Use

1. Open Teletalk application form
2. Fill manually once → **Save Form** (bottom-right panel)
3. Next job/circular → **Auto Fill**
4. Verify everything → submit yourself

Multi-step form? **Save on each page** — data merges. Settings: import/export JSON backup.

## Privacy

Local browser storage only. No server. Runs only on `*.teletalk.com.bd`. Does not submit forms or pay fees.
